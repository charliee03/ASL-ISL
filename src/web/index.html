How the repository is organized

src/recognition

Hand keypoint extraction and transformer-based ASL gloss prediction.

src/translation

LLM-backed ASL to ISL gloss translation with vocabulary validation.

src/generation

GAN generator/discriminator scaffold for avatar synthesis.

src/api

FastAPI endpoints for health, keypoints, and gloss prediction.

src/utils

Feature extraction, MediaPipe helpers, metrics, and video tools.

configs / docs / Dataset / models

Configuration, reports, datasets, and trained model assets.

Pipeline

What is live today

1. Keypoint extraction

Reads a frame, extracts 27 landmarks, and returns normalized pose and hand tensors.

2. ASL recognition

Uses the transformer classifier to predict a gloss and confidence score.

3. ASL to ISL translation

Implemented as a library module and vocabulary filter, ready for API exposure.

4. Avatar generation

GAN generator/discriminator exists as the model scaffold for future rendering.
